# Contributing to Self Growing World Simulation

> **DISCLAIMER:** This project is for **fun and educational purposes only**. Using this software assumes you agree that the creators have no liability for any generated content.

First off, thanks for taking the time to contribute!

## How to Contribute

1.  **Fork the repository** and clone it locally.
2.  **Create a branch** for your edits.
3.  **Make changes** and ensure the simulation runs locally (`npx tsx src/scripts/run-simulation.ts`).
4.  **Submit a Pull Request** describing your changes.

## Development Guidelines

- **No Commercial Use**: Do not introduce dependencies or code intended for commercial exploitation.
- **Code Style**: formatting is handled by Prettier (if configured) or standard TS conventions.
- **Testing**: Run the headless script to verify nothing broke.

## Issues

- Use the provided issue templates for bugs and feature requests.
- Be respectful and constructive.
