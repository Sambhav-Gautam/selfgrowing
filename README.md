## ðŸŒŸ Features
- **Procedural Town Planning**: A massive 200x200 grid featuring winding rivers, dense forests, and distinct residential and commercial zones.
- **Advanced Architecture**: Highly detailed, toy-like 3D models including Hospitals, Government buildings, Parks, and Mansions.
- **Social Graph Simulation**: Agents form complex relationships (Friendship, Hate, Family) and track their needs (food, safety, rest).
- **Stunning Render Pipeline**: Powered by React Three Fiber with bloom post-processing, soft shadows, and a beautiful UI glassmorphism HUD.
- **Persistent State**: History is saved to JSON and committed back to the repo automatically via GitHub Actions.
